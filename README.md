
# Evans Kumi — React + TypeScript + Tailwind Portfolio

A responsive, accessible, single‑page portfolio built with **Vite + React + TypeScript + Tailwind CSS**.

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
npm run preview
```

## Deploy

- **Vercel**: Import the repo → Framework Preset: Vite → Deploy.
- **GitHub Pages**: `npm run build` then publish the `dist/` folder or use an action.

## Tech

- React 18, TypeScript, Vite
- Tailwind CSS 3
- Mobile‑first, dark‑mode friendly (respects OS setting)
